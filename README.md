# HE Performance Platform

Landing page and PWA shell for **Clínica HE Performance**, focused on health, metabology and performance.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React
- next/image with Unsplash remote assets

## Commands

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```

## Routes

- `/`: public landing page.
- `/login`: patient login placeholder.
- `/app`: patient portal placeholder.
- `/dashboard`: medical dashboard placeholder.

## Project Structure

```text
app/
components/
docs/API.md
lib/content.ts
public/
```

## Notes

- Official logo file was not directly available from public search, so `public/logo.svg` and `public/logo-white.svg` are custom SVG marks based on the HE Performance name and premium dark/gold positioning.
- Contact and Instagram data comes from the provided project brief and public Instagram snippets.
